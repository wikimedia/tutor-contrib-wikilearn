wikilearn plugin for `Tutor <https://docs.tutor.edly.io>`__
###########################################################

wikilearn plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/wikimedia/tutor-contrib-wikilearn.git

Usage
*****

.. code-block:: bash

    tutor plugins enable wikilearn
